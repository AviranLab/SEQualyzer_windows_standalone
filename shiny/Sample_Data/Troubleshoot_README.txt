In case a text file you input does not work and throws a validation error,

1. Check your R version. Write getRversion() in R console and hit enter. If R version is <= 3.1.1, upgrade R.

2. Check your working directory. Write getwd() in R console and hit enter. If working directory is not the ‘Code’ folder of SEQualyzer, change working directory to the ‘Code’ folder.

3. Open the text file that you input. Check the file paths for correctness.

4. Make sure that the text file is formatted according to section 3 of SEQualyzer manual.
